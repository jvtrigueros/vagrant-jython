# vagrant-jython
>  A quick way to get started using Jython.

## Getting Started

Clone this repo

    git clone https://github.com/jvtrigueros/vagrant-jython.git

Initialize VM

    cd vagrant-jython
    vagrant up

Enjoy!

```sh
vagrant@vagrant-ubuntu-trusty-64:~$ /opt/jython/bin/jython
```
